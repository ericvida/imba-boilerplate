This is an Imba Boilerplate with the following features
- Imba
- Webpack 4
- Hot Module Reloading
- import 'style.sass|scss' within imba files

```
# webpack
cd imba-boilerplate
yarn add
yarn dev

```
Open up [localhost:1234](http://localhost:1234/)
